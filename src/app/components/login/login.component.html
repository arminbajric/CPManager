About app CPManager is single page application developed using Angular 7 framework,HTML5,CSS and Bootstrap for testing purposes. App is imagined as full solution for vehicle park organization and maintenance as it offer you interaction with workers. This app is designed to help managers to organize vehicle park and to be informed with lastest reports.It provides all major events and possible problems to keep eye on. Design concept has two major directions.First is to provide managers with data they neeed and second to help other workers involved to stay informed and submit their work. Because of this app will show options based on roles and their requirements.But in the end it is all conected into one single page appplication. Application is developed to use data from .json static files so it is not possible to make any changes to it or to add new data.
